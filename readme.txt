== Description ==
Use this plugin to add the new google “Are you robot?” recaptcha to wp-login page, registration page, comment forms and buddypress registration page. <br />
Get the public key and private key from  https://www.google.com/recaptcha/admin and enter the keys in plugin settings page and enable the recaptcha.
<br />
Plugin page: http://www.idiotinside.com/are-you-robot-google-recaptcha-for-wordpress/ <br />

== Installation ==


1. Upload plugin folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to use? =

Register your site in google recaptcha and enter private key and public key in the plugin settings page. <br />
See plugin settings page for more instruction.
