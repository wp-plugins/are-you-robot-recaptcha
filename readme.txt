Adds the new google “Are you robot?” recaptcha to wp-login page, registration page, comment forms, buddypress registration page and contact form 7.

== Description ==

Add the new google “Are you robot?” recaptcha to the following:

*   wp-login page
*   registration page
*   comment forms
*   buddypress registration page
*   contact form 7
